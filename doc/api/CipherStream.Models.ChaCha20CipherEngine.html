Class ChaCha20CipherEngine

ChaCha20CipherEngine class. This class implements ChaCha20 cipher. ChaCha20 is a variant of Salsa20 stream cipher. ChaCha20CipherEngine provides methods responsible for: initialization, encryption / decryption tasks and managing loger state.
